# New Code

A small web app / static site project.

## Install

Install dependencies:

```
npm install
```

## Run

Start the app (if applicable):

```
npm start
```

## Files of Interest

- `app.js`, `app.min.js` — main scripts
- `index.html` — entry page
- `styles.css` — styles
- `server/` — local server files

## Contributing

Feel free to open issues or PRs.

## Usage

Open `index.html` in a browser or serve the folder with a static server:

```
npx serve .
```

## Features

- Lightweight static site with an optional Node server in `server/`.
- Precompressed assets available (`.br` files) for efficient delivery.
